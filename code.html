Code

Rivet wouldn’t have been possible to make without many, many open source frameworks, tools, and libraries. To give back to the open source community in whatever way we can, we’ve decided to release our iOS and Android code on Github. We hope you can get something useful out of it, whether it’s seeing a useful pattern, learning more about features in an SDK, or noticing a way not to code something :P
